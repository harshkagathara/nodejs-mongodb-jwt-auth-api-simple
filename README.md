# nodejs-mongodb-jwt-auth-api
Build Secure (JWT) Token Based Authentication API with Node
<br>

```
 npm install
 npm run dev
```

<b>URL :</b><a href="http://localhost:3000/signup">http://localhost:3000/signup</a>
<br>
<b>variable: </b> username, email, password  <br>
